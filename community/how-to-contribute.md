# 🤝 How to Contribute

This repository is a community-driven space for early‑career designers.

## 📝 Add Opportunities
- Submit a pull request with new roles or programs  
- Include links and short descriptions  

## 📁 Share Resources
- Portfolio tips  
- Case study templates  
- Accessibility or product strategy guides  

## 🎨 Showcase Your Work
- Figma prototypes  
- Design systems  
- UX case studies  

## 💬 Join the Conversation
- Open issues to ask questions or share advice  
- Use discussions (if enabled) to connect with others  

This hub grows stronger with every contribution.
